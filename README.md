
# TrabalhofLP1Bonini

## Descrição

Este sistema em linguagem C foi criado para o gerenciamento de playlists de músicas. Ele conta com um arquivo de músicas disponíveis, permitindo o ususário visualizar, buscar, adicionar e remover músicas de playlists, além de ordená-las por nome ou duração. Fora isso o sistema salva as playlists em arquivo para que as informações não sejam perdidas ao encerrar o programa.

- **4**. `utils.h` e `utils.c` - Funções auxiliares.
